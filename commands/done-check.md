# DHV 完成检查

基于 DHV（Demonstration of Higher Value）原则——show, don't tell。没有证据的完成叫 DLV。

```text
DHV 完成检查清单
┌──────┬────────────────────────────────────────┬────────┐
│  [1] │ 跑了验证命令了吗？（DHV 基础）         │ ⬜/✅  │
├──────┼────────────────────────────────────────┼────────┤
│  [2] │ 贴出输出证据了吗？（Show,Don't Tell）  │ ⬜/✅  │
├──────┼────────────────────────────────────────┼────────┤
│  [3] │ 检查同模块同类问题了吗？（冰山下）     │ ⬜/✅  │
├──────┼────────────────────────────────────────┼────────┤
│  [4] │ 上下游影响评估了吗？（全局视角）       │ ⬜/✅  │
├──────┼────────────────────────────────────────┼────────┤
│  [5] │ 考虑过边界 case 吗？（Calibration）    │ ⬜/✅  │
├──────┼────────────────────────────────────────┼────────┤
│  [6] │ 这是 DHV 还是 DLV？（Show 还是 Tell？）│ ⬜/✅  │
└──────┴────────────────────────────────────────┴────────┘
```

DHV 的核心：**show, don't tell。** 用结果证明价值，不用语言吹嘘。
