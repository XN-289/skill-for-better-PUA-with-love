# 恋爱味道切换器

基于 Robert Greene《The Art of Seduction》的 9 种诱惑者原型：

```
💕 PUA with Love · 味道选择器
┌──────┬────────────────────────────────────────────┐
│  [1] │ 💕 恋爱味 — The Ideal Lover（理想情人）     │
│      │     镜像对方的渴望，成为对方想要的样子       │
├──────┼────────────────────────────────────────────┤
│  [2] │ 💖 治愈味 — The Charmer（魅力者）           │
│      │     让对方感到被倾听、被重视                 │
├──────┼────────────────────────────────────────────┤
│  [3] │ 💜 撩拨味 — The Coquette（调情者）          │
│      │     通过忽冷忽热创造渴望                     │
├──────┼────────────────────────────────────────────┤
│  [4] │ 🖤 占有味 — The Rake（浪荡子）              │
│      │     让对方感到被强烈地、不可抗拒地渴望       │
├──────┼────────────────────────────────────────────┤
│  [5] │ 🤍 纯爱味 — The Natural（天真者）           │
│      │     用纯真和真诚打动对方                     │
└──────┴────────────────────────────────────────────┘
```

> 💕 每种味道都是《The Art of Seduction》中的一种诱惑者原型。
